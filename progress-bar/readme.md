<h3>3. 이벤트 컨트롤 </h3>

<br/>

- 스크롤링 이벤트를 throttle 함수의 콜백 함수로 담아, 일정한 시간 간격으로 스크롤링 이벤트가 작동하도록 구성했습니다. 리스트가 많아지면, 무한 스크롤 이벤트가 발생할 수 있기 때문에, 이를 가정하여 throttle 함수에 담았습니다.
